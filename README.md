# How to run the project locally?
For running this project locally ,make sure that you have node.js installed in your device.

# Steps for run locally: 

### 1. Clone the repository (https://github.com/Firoz671/assesment-task.git) or download the zip file
### 2. Open the project file in vs code
### 3. Download all necesary dependencies (listed in package.json)
### 4. Install npm using "npm install"
### 4. In terminal type  "npm run dev" then project can be show locally
